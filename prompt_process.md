# 📊 课堂行为智能分析系统全流程工作流报告
本系统基于 OpenClaw 平台，深度整合了 YOLOv8 计算机视觉算法 与 多模态大模型 (VLM) 技术，实现了从原始教学视频到专业科研级分析报告的全自动化转化。

## 🛠 一、 系统架构四阶段
### 1. 视频流预处理 (Video Pre-processing)
支撑技能：**video-processor**
核心工具：FFmpeg
输入：原始课堂全量视频 (MP4/MOV)
工作内容：
精准抽帧：按固定频率（如 5s/帧）或教学环节进行高清抽帧。
质量对齐：对图像进行标准化处理，优化体积以符合大模型最佳输入要求。
输出：标准化视频帧序列。
### 2. 学生目标定位与追踪 (Detection & Tracking)
支撑技能：**student-detector**
核心技术：YOLOv8 + OpenCV
工作内容：
YOLOv8 检测：利用 YOLOv8 深度学习模型，在每一帧中实时锁定所有学生，生成像素级边界框（Bounding Box）。
智能 ID 追踪：通过质心追踪算法实现跨帧 ID 绑定，确保 Student_01 的数据连贯性。
自动化截取：利用 OpenCV 库根据坐标点自动裁剪出以学生为中心的个体图像。
输出：按学生 ID 物理分类的个体追踪图库。
### 3. 多模态行为语义识别 (Behavior Semantic Analysis)
支撑技能：**behavior-analyzer**
核心技术：通义千问 Qwen-VL-Max + 配置化 Prompt
输入：个体追踪图库
工作内容：
VLM 深度推理：通过 indicators.yaml 驱动的 Prompt，单次调用同时识别：
专注度：抬头/低头状态判断。
认知加工：书写/笔记行为判定。
情感状态：6 种情绪分类（专注、困惑、愉悦、无聊、紧张、中性）。
量化评分：执行规范级算法：参与度 = 抬头(30%) + 举手(30%) + 书写(20%) + 积极情绪(20%)。
输出：video_analysis_report.json (系统核心数据中台)。
### 4. 规范化看板生成与 AI 决策 (Visualization & Reporting)
支撑技能：**behavior-visualizer**
核心技术：Python Matplotlib / Seaborn + AI 文本生成
工作内容：
宏观趋势图：生成“三线合一”趋势图（专注-认知-情感），反映课堂节奏。
个体对比图：生成“左优右差”双子图，直观揭示表现极端个体的行为差异。
AI 智能总结：基于数据自动生成课堂表现评价（如：优秀/良好/一般）及教学改进建议。
输出：index.html 交互式看板报告。
## 🚀 二、 系统核心亮点
快慢模型架构 (YOLO + VLM)：

由 YOLOv8 负责高频、低功耗的空间定位。
由 Qwen-VL 负责低频、高精度的语义理解。
价值：极大提升了分析效率，同时将 API 调用成本降低了 70% 以上。
配置化指标库 (Decoupled Config)：

识别指标与代码解耦，仅需修改 indicators.yaml 即可快速适配科学、体育等不同学科的评价体系。
科研级可视化诊断：

系统不再只输出简单百分比，而是通过“个体行为时序对比”和“多维关联分析”，为教师提供具备“诊断性”的教学反馈。
